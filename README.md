# CS2-Terminal-Deal-Calculator
A python and HTML/CSS application that screenshots your screen to calculate if the deal being offered to you by the arms dealer is a good deal or not.

## Installation
### Tesseract-OCR
**Tesseract-OCR** can be easily downloaded from this url (https://github.com/UB-Mannheim/tesseract/wiki). Make sure that you do NOT change the default install directory, and leave it how it is unless you are planning to update the code yourself to the new directory.
### The app
After installing **Tesseract-OCR,** you should be able to download the executable file, and run it. Alternatively, you can download the source code, install all the python packages required, and * *then* * you will be able to edit the code, run, and recompile it to your liking.

## Exiting Application
Make sure that you close the application by pressing **Ctrl+C** inside of the terminal, otherwise the process will continure to run in the background, not that it will eat resources, but it will keep runnng.
